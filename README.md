# portfolio
Ceci est un projet backoffice pour une api de gestion de projet personnel

Objectif du projet
Ce projet a pour but de fournir une base technique moderne et évolutive pour créer des portfolios dynamiques. Chaque utilisateur peut installer sa propre instance du back-end, ajouter ses projets, les modifier, les organiser, et connecter n’importe quel front-end.

Le projet est pensé pour être :

Facile à déployer en fournissant un docker-compose
Facile à comprendre en fournissant une documentation avec mkdocs
Facile à personnaliser
Facile à contribuer vu que le projet est opensource et sous licece MIT
